  1. Зайти на віртуалку, там буде обмежений користувач.

  2. Встановити nmap tcpdump, глянути який відкритий порт (який роздає текст) (nmap не буде встановлений, встановити буде важко через поломані DNS сервери. sudo-Доступ до nmap, tcpdump, dnf, vim буде. In ProgressТам буде ip+port, user, password для sFTP сервер.

  3. Заходим на FTP, там буде заархівована файл-файлова система, яку треба буде змонтувати як loop пристрій. (малий ttl; no default route)  Розархівовуємо, монтуємо, буде папка з купою файлів-папок.Завдання через find знайти саме .key (можливо дати точніші параметри, як розмір або вміст або права доступу або власник/група.), скопіювати в домашню папку.  

    Це текст 3.

  4. ssh -i file.key user2@localhost.

    Це текст 4.

  5. Через fsync (без примінення змін) знайти яким файлом відрізняються дві директорії, прочитати цей файл, у ньому або пароль або якась інфа, яку треба надіслати по TCP. 

    Це текст 5.

    Це текст 6.
